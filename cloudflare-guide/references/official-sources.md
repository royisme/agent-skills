# Cloudflare Official Documentation Sources

## LLM-Optimized Documentation URLs

Cloudflare provides these URLs specifically for LLM consumption.

### Primary Sources

| Source | URL | Topics Covered |
|--------|-----|----------------|
| **Workers** | `https://developers.cloudflare.com/workers/llms-full.txt` | Worker development, Wrangler config, bindings, runtime APIs, deployment |
| **Workers Quick** | `https://developers.cloudflare.com/workers/prompt.txt` | Condensed reference for common patterns |
| **Pages** | `https://developers.cloudflare.com/pages/llms-full.txt` | Static sites, Pages Functions, build config, Git integration |
| **Developer Platform** | `https://developers.cloudflare.com/developer-platform/llms-full.txt` | D1, R2, KV, Queues, Hyperdrive, Durable Objects, AI Gateway |
| **Agents SDK** | `https://developers.cloudflare.com/agents/llms-full.txt` | AI agents, state management, WebSocket, scheduling |
| **Platforms** | `https://developers.cloudflare.com/cloudflare-for-platforms/llms-full.txt` | Workers for Platforms, multi-tenant, dispatch namespaces |
| **Full Index** | `https://developers.cloudflare.com/llms.txt` | Directory of all available docs |
